Detailed Description

Represents each cluster generated in the process. It holds a cluster identifier, a pointer to a vector of members, the maximum distance within members of the cluster, the cluster centroid (member with smallest total distance to other members), the cluster radius and a boolean flag that indicates whether or not the cluster has been merged into a posterior cluster.
